ESM and TypeScript rewrite of [npm/ini](https://github.com/npm/ini).

## Features

- Rewritten in ESM and TypeScript, ships with type declarations
- Refactor all to pure functions, fully tree-shakable

## Install

```shell
npm install ini-es
```

```shell
yarn add ini-es
```

```shell
pnpm add ini-es
```
